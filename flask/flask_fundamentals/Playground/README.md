# Playground
A dynamic web application built with Flask that generates a customizable flex colored boxes based on URL parameters.

### How to work 
1. Activate the environment:
    - flask_env\Scripts\activate

2. Install dependencies:
    - pip install flask

3. Run the application:
    - python play.py

4. Access the routes:
    - http://127.0.0.1:5000/play (Default: 3 lightblue boxes)
    - http://127.0.0.1:5000/play/7  (7 lightblue boxes)
    - http://127.0.0.1:5000/play/4/green  (10 green boxes)
